       Security & encryption 
      

        
This panel serves plain HTTP by default — anyone on the network can read the traffic. Encrypt it with either option below. Both are optional and off until you turn them on.


        
        

          
Tailnet · Tailscale…

          
Joins an encrypted WireGuard mesh so you can reach this node — including over the internet — with all traffic encrypted. Optionally fronts the UI with a real HTTPS certificate (browser padlock, nothing to install).

          
Tailscale is not installed on this node.

Self-signed HTTPS…

          
Encrypts LAN traffic with no dependencies and nothing to install. Your browser shows a one-time “not secure” warning (there is no trusted root) — that is expected for a self-signed certificate; the connection is still encrypted.

       Miracast 
      

        

        
        
        
auto-stop aftermin (0 = never)

        
abort if uplink losts (0 = off)
          

        
Miracast is Wi-Fi Direct — on a single-radio node it can knock the box off its own network. Auto-abort restores the last mode; a time-box bounds how long it holds the radio. Best of all: run the uplink on Ethernet, or pin P2P to a 2nd adapter above.
